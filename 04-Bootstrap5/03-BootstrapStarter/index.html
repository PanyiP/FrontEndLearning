Bootstrap 5 Simple Starter

This is a boilerplate for creating and customizing Bootstrap 5 projects.

Look at "bootstrap/scss/_variables.scss" to see the variables that you can overwrite in your "scss/bootstrap.scss" file. For any custom styles, add them to the "scss/styles.scss" file. Do not edit the CSS files directly.

Update Versions

  1. Download latest Bootstrap and FontAwesome files.
  2. Bootstrap: Update "js" and "scss" files in the "bootstrap" folder.
  3. FontAwesome: Update "js" and "scss" files in the fontawesome folder.
  4. FontAwesome: Update files in the "webfonts" folder.
